# 🛒 ShopEase - Responsive E-Commerce Website

## 📌 Project Overview

ShopEase is a responsive e-commerce website developed using **HTML5**, **CSS3**, and **JavaScript** as part of the **Web Development Internship Project**.

The project demonstrates the fundamentals of front-end web development by creating a modern, interactive, and user-friendly online shopping interface.

---

## 🚀 Features

- Responsive design for desktop, tablet, and mobile devices
- Modern navigation bar
- Hero section
- Product categories
- Product cards with images
- Product names, prices, and descriptions
- Category labels
- Add to Cart button
- Cart item counter
- Show/Hide product details
- Search products
- Contact form with validation
- Smooth scrolling
- Scroll-to-top button
- Professional footer
- Hover effects and animations

---

## 🛠 Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla)
